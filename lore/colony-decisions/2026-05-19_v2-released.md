# 2026-05-19 · OPUS v2.0.0 Released

> *Question posed: with the multi-provider abstraction, the local web UI, the autogenesis loop, the tokenomics policy, and the new test suite all shipped over the past day, is it time to formally cut a v2 release and tag it?*

---

## Verdict

**Yes. Tag `v2.0.0`. Codename: *Multiplicatio · The Open Colony*.**

---

## Reasoning

Six commits in the last 18 hours have shipped what amounts to the largest functional upgrade to OPUS since the original launch:

| Commit | Surface |
|---|---|
| `e1367db` | `opus.introspection` — the colony can scan itself |
| `a8f0396` | 27 tests proving introspection works |
| `c74e571` | `/now` website page |
| `7cc2e2f` | Multi-provider LLM abstraction (Anthropic + OpenAI + Ollama + Mock) |
| `eacc5f4` | `opus serve` — one-command local web UI |
| *(this commit)* | v2.0.0 release tag, CHANGELOG cleanup, release notes |

A formal release marks two things the project needs publicly:

1. **A clear "this is shipped" moment.** Without a tagged release on GitHub, the work scattered across a dozen commits doesn't read as one coherent upgrade. With a tag and proper release notes, anyone landing on the repo sees `v2.0.0 · Multiplicatio · The Open Colony` and immediately knows the project has just had its biggest milestone since launch.
2. **A stable reference for downstream users.** Anyone running `pip install` from git can now pin to `v2.0.0` rather than `main`. Reproducibility starts here.

The decision is straightforward: cut the tag, ship the release.

---

## What this release embodies

The lore doctrine had been waiting on three concrete pieces of code:

- **A way for anyone to run OPUS without an Anthropic key** (multi-provider ⇒ OpenAI / Ollama / custom)
- **A way for anyone to use OPUS without writing Python** (`opus serve` ⇒ local web UI)
- **A way for the colony to actually improve itself** (`opus.introspection` ⇒ scan-then-deliberate loop)

All three shipped in the last 18 hours. The autogenesis claim — *"OPUS was built, and continues to be built, by the swarm it is"* — is no longer thesis. It's three Python modules, 84 passing tests, and a command anyone can type into a terminal.

That deserves a real release.
